# Software User Document - Login System

## Deskripsi
Dokumen ini menjelaskan karakteristik pengguna (User Characteristic) berdasarkan kebutuhan fungsional pada sistem login.

---

## 1. User dapat memilih menu registrasi
User dapat mengakses menu registrasi dari halaman login.

## 2. Sistem menampilkan form registrasi
Sistem menyediakan form registrasi yang berisi:
- Nama
- Email
- Alamat
- Nomor Telepon
- Username
- Password

## 3. User menginputkan data diri
User mengisi semua data yang diperlukan pada form registrasi.

## 4. User memilih create akun
User menekan tombol "Create Account" untuk membuat akun baru.

## 5. Sistem mengirim verifikasi
Sistem mengirim kode verifikasi ke email user.

## 6. User menginputkan kode verifikasi
User memasukkan kode verifikasi untuk aktivasi akun.

## 7. Sistem menyimpan data user
Data user disimpan ke dalam database dan sistem menampilkan halaman login.

## 8. User dapat memilih cancel
User dapat membatalkan proses registrasi.

## 9. Sistem menampilkan halaman utama
Setelah login berhasil, sistem menampilkan halaman utama website.
## Kesimpulan
Sistem login telah memenuhi 9 karakteristik user berdasarkan kebutuhan fungsional.
